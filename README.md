# Heart Disease Model Evaluation

This project demonstrates training and evaluating a Logistic Regression model using the Heart Disease dataset.

## Objectives
- Split dataset into training and testing sets
- Train a Logistic Regression model
- Predict on test data
- Evaluate using accuracy, precision, recall
- Visualize confusion matrix

## Dataset
- Heart Disease Dataset

## Model Used
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab
